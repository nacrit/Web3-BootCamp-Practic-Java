## 模拟实现最小的区块链原理
> 用自己熟悉的语言模拟实现最小的区块链， 包含两个功能：
> POW 证明出块，难度为 4 个 0 开头
> 每个区块包含previous_hash 让区块串联起来。
> 如下是一个参考区块结构：
> block = {
> 'index': 1,
> 'timestamp': 1506057125,
> 'transactions': [
> { 'sender': "xxx",
> 'recipient': "xxx",
> 'amount': 5, } ],
> 'proof': 324984774000,
> 'previous_hash': "xxxx"
> }

代码实现：[Demo3MiniBlockChain.java](aa_base/src/main/java/com/nacrt/base/Demo3MiniBlockChain.java)，运行main函数即可
